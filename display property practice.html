The css display property

Display" inline

any height and width properties have to effect on an inline element. by default inline elements display on the same line as other inline elements. Inline elements are only as wide as they need to be, i.e, they are only as wide as their content

Display:inline block

Like a block element, you can set the Height and width of an inline-block element. Inline-block elements, through, display on the same line, just like an inline element.

Display: block

block elements are the opposite of inline elements. Remember that inline elements display on the same line. Block elements, however, start on a new line and will take up the full width of their parent container. commonly usedblock elements include:
  • div
  • h1 - h6
  • p
  • li
